Pengenalan HTML

HTML (Hypertext Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat halaman web dan menampilkan berbagai informasi di dalam browser. HTML terdiri dari tag-tag yang menginstruksikan browser untuk menghasilkan tampilan sesuai dengan yang diinginkan.

Sejarah HTML

  • HTML 2.0 (RFC1866) disetujui pada 22 September 1995
  • HTML 3.2 - 14 Januari 1996
  • HTML 4.0 - 18 Desember 1997
  • XHTML 1.0 - 24 Januari 2000

Struktur Dasar HTML

Sebuah dokumen HTML memiliki struktur dasar sebagai berikut:

<html>
<head>
  <title>Judul Halaman</title>
</head>
<body>
  ... Konten halaman ...
</body>
</html>

Tag-Tag Dasar

Berikut beberapa tag dasar dalam HTML:

Heading: <h1>...</h1> hingga <h6>...</h6> untuk membuat judul.
Paragraf: <p>...</p> untuk membuat sebuah paragraf.
Gambar: <img src="gambar.jpg" alt="Deskripsi Gambar" /> untuk menampilkan gambar.
Hyperlink: <a href="url">Teks Link</a> untuk membuat tautan.
